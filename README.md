# FoodMapVue

前後端分離專案。前端使用 Vue 3，後端使用 Spring Boot，資料庫使用 MSSQL。功能包含會員、店家、搜尋、預約、客服、購物等模組。

## 功能

- 會員系統：註冊 / 登入 / 權限控制
- 預約管理：新增 / 修改 / 刪除 / 衝突檢查
- 合約管理：CRUD 與提醒判斷
- 黑名單 / 評論管理
- API 整合：前端、金流、郵件、地圖等

## 技術棧

- Vue 3 + Vite
- Vue Router
- Axios
- Bootstrap / 自訂樣式

## Database

MSSQL
